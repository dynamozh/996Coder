## 2.Car Plate Recognition

### 安装包

`python -m pip install hyperlpr`

### Linux/Mac 编译

- 仅需要的依赖OpenCV 3.4 (需要DNN框架)

```bash
cd Prj-Linux
mkdir build 
cd build
cmake ../
sudo make -j 
```
